# Modern Neumorphic Portfolio

This repository contains the source code for a modern, minimalist, single-page portfolio designed for developers and creatives. It is built with React and Tailwind CSS, featuring a clean neumorphic design, light/dark themes, and smooth animations.

---

## About The Project

This project is a personal portfolio template designed to showcase skills, work experience, education, and projects in a visually appealing and interactive way. The design is based on the neumorphism trend, giving it a soft, clean UI. It's built to be fully responsive and easily customizable.
